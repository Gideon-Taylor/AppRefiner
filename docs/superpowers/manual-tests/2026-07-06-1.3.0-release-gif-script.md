# 1.3.0 Highlight GIF — Recording Script

## Starter code (paste into a PeopleCode editor before recording)

```peoplecode
/* Order Pricing Calculation */
Local string &stateCd = "CA";
Local number &subtotal = 1249.99;
Local number &taxRate, &shipping, &total;

If Upper(&stateCd) = "CA" Then
   &taxRate = 0.0925;
Else
   If Upper(&stateCd) = "NY" Then
      &taxRate = 0.08875;
   Else
      If Upper(&stateCd) = "TX" Then
         &taxRate = 0.0625;
      Else
         &taxRate = 0.0;
      End-If;
   End-If;
End-If;

If &subtotal > 1000 Then
   &shipping = 0;
Else
   &shipping = 15.99;
End-If;

&total = &subtotal + (&subtotal * &taxRate) + &shipping;
```

No Record/Field/Component references on purpose — only local variables, parameters, and builtin functions, so nothing shows up as a type/metadata error in an environment that doesn't have these objects defined. `Upper(&stateCd)` appears 3x on purpose — feeds the "replace all occurrences" part of step 3.

## Shot list

1. **AppRefiner + Instances tab** — show the main window, open Instances tab, point at DB/Tools version/open editors for a session, bring it to front to transition into Application Designer.
2. **Baseline save** — paste starter code, `Ctrl+S` (snapshot #1).
3. **Extract Local Variable** — select `Upper(&stateCd)` in the first `If`, run Extract Local Variable, name `&normalizedState`, replace all 3 occurrences. `Ctrl+S`.
4. **Extract Function/Method** — select the whole `If &normalizedState = "CA" ... End-If;` chain, run Extract Function/Method, name `GetTaxRateForState`. Confirm detected input (`&normalizedState`) / return (`&taxRate`). Call site collapses to `&taxRate = GetTaxRateForState(&normalizedState);`. The new function must land above the program's top-level statements (PeopleCode requires `Function...End-Function` defs to precede the main executable body) — if it gets inserted at the bottom instead, that's a bug worth flagging. `Ctrl+S`.
5. **Convert If ↔ Evaluate** — inside the new function, cursor in the If/Else-If chain, run Convert If ↔ Evaluate → becomes `Evaluate &normalizedState / When "CA"/"NY"/"TX" / When-Other`. `Ctrl+S`.
6. **Autocomplete → Message Catalog → Capture-Result Quick Fix** — back in main body, type `If &total <= 0 Then`, Enter. Type `Msg`, `Ctrl+Space`, pick `MsgGetText`, type `(` for the call-tip. Inside the parens, `Ctrl+Space` again to open the Message Catalog browser, pick a message, let it fill in set/number/text. Close `);` — unassigned result gets squiggled. Cursor on squiggle, `Ctrl+.` → capture-result quick fix wraps it in `Local string &_ = MsgGetText(...);`. `End-If;`. `Ctrl+S`.
7. **Snapshot History (closer)** — open the redesigned Snapshot History dialog, scroll the 5 saves, pick baseline vs. current, show side-by-side diff with per-row change counts, click copy-diff once. End here.

## Expected final form (after all 6 steps)

```peoplecode
Function GetTaxRateForState(&normalizedState As string) Returns number
   Local number &taxRate;
   Evaluate &normalizedState
      When "CA"
         &taxRate = 0.0925;
      When "NY"
         &taxRate = 0.08875;
      When "TX"
         &taxRate = 0.0625;
      When-Other
         &taxRate = 0.0;
   End-Evaluate;
   Return &taxRate;
End-Function;

/* Order Pricing Calculation */
Local string &stateCd = "CA";
Local number &subtotal = 1249.99;
Local number &taxRate, &shipping, &total;

Local string &normalizedState = Upper(&stateCd);
&taxRate = GetTaxRateForState(&normalizedState);

If &subtotal > 1000 Then
   &shipping = 0;
Else
   &shipping = 15.99;
End-If;

&total = &subtotal + (&subtotal * &taxRate) + &shipping;

If &total <= 0 Then
   Local string &_ = MsgGetText(1, 1, "Order total must be greater than zero.");
End-If;
```

(`MsgGetText` set/number/text are placeholders — the real catalog browser selection during recording will fill in whatever you actually pick from your live message catalog.)

## Recording tips
- Bump font size a couple points above daily-driver size in both apps — GIF compression eats detail.
- Pause ~1-2s after each result so it reads instead of blurring past.
- Record as video (ScreenToGif / Game Bar) and convert to GIF after — easier to trim dead air.
- Consider splitting into shorter clips (refactoring toolkit = steps 3-5, message catalog = step 6, snapshots = step 7) to drop inline next to matching bullets in the release notes, rather than one long GIF.
